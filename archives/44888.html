markdown是什么

2020-09-07 Views

  2. 不足
文章目录

  2. 不足

markdown是一种标记语言，你可以用它给文本添加格式。可能你要问了，有word等编辑器不是更好吗？

  1. markdown没有word这么多格式，但它不用你在打字的时候用鼠标进行添加格式，只需要打几个标记就行了。这可以增加打字效率。

  2. 它的应用很广，只要网站支持，你就能发有格式的评论。文章、邮件也不在话下。就算没有，写好之后，可以用一些在线网站转换

Markdown Guide的文档是这样说的

  • Markdown can be used for everything. People use it to create websites, documents, notes, books, presentations, email messages, and technical documentation.
  • Markdown is portable. Files containing Markdown-formatted text can be opened using virtually any application.
  • Markdown is platform independent. You can create Markdown-formatted text on any device running any operating system.
  • Markdown is future proof.
  • Markdown is everywhere.

markdown语法例子如下：

‘# 代表一级标题’

‘## 代表二级标题’

‘** 加粗 **’

‘~ 删除线 ~’

更多例子，不一一细说，网上教程一搜一大堆

typora

markdown是随便什么编辑器都可以写，只要把文件名后缀改成md就好。不过，仍然推荐使用对应的编辑器，可以查看它渲染后的样子。这里主要推荐typora。

因为有一些软件，虽说可以渲染，但是它左右分屏。也就是左边是原文，右边是渲染后的样子，看着难受。而typora与其它软件不同，用它写markdown，可以所见即所得。

markdown是使用css控制样式的。也就是说，你可以自由地用css定义样式，设计自己的一套排版。而普通的人，也不必自己设计，它本身就有自带的主题。而且还有主题中心，可以获得更多主题。

也只有像我这样写文章的自媒体，才需要根据自己的需求自定义。只要设计出来，以后发文章也就花几分钟进行排版，然后复制粘贴到各个平台就好了。

有文件栏，很方便的，对文件进行分类。所以说基本编程内都有这种功能

不足

markdown的缺点就是图片只能用网上的链接，所以你需要建一个自己的图床。而且你可能会发现每次先把本地图床上传，然后再复制链接显得麻烦。所以此时要使用另一种工具picgo。进入typora的设置，找到下图的位置，对其设置。没有下载picgo的要先下载，还要配置好图床，具体的看picgo的官方文档。
